# Traffic Sign Detection Project

## Overview
This project presents a system for detecting traffic signs in images using artificial intelligence.  
The system is based on a YOLO object detection model and highlights detected traffic signs using bounding boxes.

This project was developed as part of an Autonomous Vehicles course.

---

## Features
- Detects traffic signs in images
- Uses a YOLO deep learning model
- Displays bounding boxes around detected signs
- Works on real-world road images

---

## Technologies Used
- Python
- OpenCV
- Google Colab
- YOLO (You Only Look Once)

---

## How It Works
1. An image is uploaded to the system  
2. The model processes the image  
3. Traffic signs are detected  
4. Bounding boxes are drawn around detected signs  
5. The result is displayed  

---

## Example Results
The system performs well when traffic signs are clearly visible.  
However, performance decreases when signs are small, far away, or partially hidden.

---

## Limitations
- Not all traffic signs are detected
- False detections may occur
- Performance depends on image conditions such as lighting and distance

---

## Conclusion
This project demonstrates how artificial intelligence can be used to detect traffic signs.  
It highlights both the capabilities and limitations of real-world AI systems.
